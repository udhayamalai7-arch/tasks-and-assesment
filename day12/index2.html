Float + Clearfix Example

Title next to a floated image

This paragraph flows beside the image because the image is floated. The clearfix applied to the container (.clearfix) ensures the parent expands to contain the floated element, preventing layout collapse (e.g., overlapping the next section).

On narrow screens the media__img stops floating and becomes full-width above the text thanks to the media query — which is generally friendlier for mobile users.
